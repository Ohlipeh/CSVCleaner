# CSV Cleaner – Python Data Cleaning Automation

CSV Cleaner is a simple Python automation script that cleans raw CSV files and exports a structured Excel spreadsheet.

## ✅ Features

- Removes duplicate rows
- Removes empty values
- Sorts data by a chosen column
- Exports clean output to Excel (.xlsx)

## 🛠 Tech

- Python
- pandas
- openpyxl

## 📂 Structure

csv-cleaner/

- data/dados.csv
- output/clean_data.xlsx
- src/main.py

## ▶ How to Run (Windows)

1. Create venv:
   python -m venv venv

2. Activate:
   venv\Scripts\activate

3. Install:
   pip install -r requirements.txt

4. Run:
   python src/main.py
